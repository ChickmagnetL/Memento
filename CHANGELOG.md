# Changelog

All notable changes to Memento are documented in this file.

## Unreleased

## [0.1.1] - 2026-07-17

### Fixed

- The home page now refreshes active video-processing status automatically, so completed and failed jobs appear without a manual reload.
- The Windows uninstaller now stops Memento runtime processes and reliably removes application data, services, and cache files, including locked files after restart when needed.

### Documentation

- Simplified and synchronized the Chinese and English FAQ layout, and added license and acknowledgements information to the Chinese README.
